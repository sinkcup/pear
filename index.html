sinkcup PEAR Channel

Registering the channel: 
pear channel-discover sinkcup.github.io/pear
Listing available packages: 
pear remote-list -c sinkcup
Installing a package: 
pear install sinkcup/package_name
Installing a specific version/stability: 
pear install sinkcup/package_name-1.0.0
pear install sinkcup/package_name-beta
Receiving updates via a feed: 
http://sinkcup.github.io/pear/feed.xml
